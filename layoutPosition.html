盒子模型

页面元素大小，边框，与其他元素的距离
页面上的任何元素都可以看做是盒子,一个盒子由里面的内容-content,及其内容的宽度-width,高度-height,padding-内边距,border-边框,margin-外边距组成。
padding-top 上内边距，padding-right 右内边距，padding-bottom 下内边距，padding-left 左内边距
border，margin 也是类似的有上右下左四个属性。
padding:top right bottom left
简写形式：padding：1px; 表示上右下左四个方向的取值一样
padding: 1px 2px; 表示上下方向取值一样，左右取值一样
margin 也同上类似; margin 在垂直上有个合并，取两个元素margin值最大的一个。
margin:0 auto; 可以起到一个水平居中的效果。

一个盒子的实际宽度、高度由content+padding+border+margin

border-width 表示边框粗细，可选值像素大小px，thin细，medium中，thick粗 border-style 表示边框样式，可选值：dashed 虚线，dotted 点，solid 实线，double 双实线 border-color 颜色 往往可以简写成border:width style color

定位机制

文档流，浮动定位，层定位

默认文档流定位，从上到下从左到右；
浮动定位，使用 float 属性设置，左浮动、右浮动，不浮动
层定位，使用 position 属性设置

文档流定位

float 的可选值有 block，inline 和 inline-block，可以用 display 属性进行元素类型转换。 block 属性的元素要独占一行，元素的 height、width、margin、padding 属性都可以设置。 具有 block 属性的常见元素有<div>、<p>、<h1...6>、<ol>、<ul>、<table>、<form>；其他非 block 类型的元素可通过 display 属性设置成 block。inline 类型元素不用独占一行，高度和宽度不可变，width 就是它包含的文字或图片的宽度，不可改变， 具有此类型的元素有:<span> 和 <a>，其他非inline类型的元素同样可以通过用 display: inline;设置成inline类型；inline 属性默认元素之间带有间隙。 inline-block 类型元素同时具有 inline 和 block 的特点，不独占一行，但是高度宽度内外间距都可以设置， 具有 inline-block 属性的元素有<img>。当 display 取值为 none 则元素不显示。

层定位

当我们希望一个网页元素可以层叠在另外一个网页元素的上面时，可以考虑用层定位。 层定位先用 position 指定定位参考类型，可选值：fixed 固定定位，relative 相对定位，absolute 绝对定位； 接着用 left-与上级盒子左边框的距离，right-与上级盒子右边框的距离，top-与上级盒子上边框的距离， bottom-与上级盒子底边框的距离，z-index 用于起到层的定位，值大的在上面。

position 属性默认值为 static ，属于文档流，此时 top、bottom、left、right 统统无效。 fixed 参照物是浏览器窗口， relative 参照物是其直接父元素， absolute 参照物是 static 外的第一个（不一定是直接父元素）父元素。
